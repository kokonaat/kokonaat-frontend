# Bengali / English Retail Term Glossary

Consistent terminology used across Kokonaat admin UI and exports.

| English | Bengali | Context |
|---------|---------|---------|
| Receivable | পাওনা | Money owed to the shop by customers |
| Payable / Due | দেনা | Money owed by the shop to vendors |
| Stock / Inventory | মজুদ | Product quantity on hand |
| Sale | বিক্রয় | Outgoing transaction type |
| Purchase | ক্রয় | Incoming stock transaction |
| Payment | পেমেন্ট | Cash/bank payment recorded |
| Commission | কমিশন | Commission transaction type |
| Customer | গ্রাহক | Business partner (buyer) |
| Vendor | সরবরাহকারী | Business partner (supplier) |
| Ledger | খাতা | Customer/vendor account statement |
| Balance Sheet | ব্যালেন্স শিট | Financial summary report |
| Expense | খরচ | Operating costs |
| Transaction | লেনদেন | Any financial record |
| Transaction Board | লেনদেন বোর্ড | Main transactions page |
| UOM (Unit of Measure) | পরিমাপ একক | kg, piece, etc. |
| Rows per page | প্রতি পৃষ্ঠায় সারি | Table pagination |
| Dashboard | ড্যাশবোর্ড | Home analytics page |
| Reports | রিপোর্ট | Report module |
| Shop | দোকান | Retail store entity |
| Cash | নগদ | Payment method |
| Bank Transfer | ব্যাংক ট্রান্সফার | Payment method |
| Mobile Payment | মোবাইল পেমেন্ট | Payment method |
| Subtotal | উপমোট | Report/export totals |
| Grand Total | সর্বমোট | Report/export totals |
| Pending | বাকি | Unpaid balance |
| Paid | পরিশোধিত | Settled amount |
| Opening Balance | প্রারম্ভিক ব্যালেন্স | Ledger start balance |
| Closing Balance | সমাপনী ব্যালেন্স | Ledger end balance |

## Notes

- Bengali UI uses **LTR** layout (no RTL changes).
- API error messages remain in **English** (backend-owned).
- User-entered data (names, remarks, notes) is **not translated**.
- PDF exports embed **Noto Sans Bengali** when locale is `bn`.
